  • makePaymentView(clientToken:category:completion:)

    Factory method to make a KlarnaPaymentView.

    Note

    Remember to set KlarnaPaymentView‘s delegate when you get a loaded KlarnaPaymentView back, if you want to listen to the delegate methods.

    Klarna payment view will be initialized with frame .zero, auto layout is the recommended way to manage the payment view.

    KlarnaPayments.makePaymentView(clientToken: "token", category: "pay_now") { (result) in
    switch result {
    case .success(let paymentView):
        // Do something with the payment view, maybe set the delegate first
        // paymentView.delegate = `some object`
    case .failure(let error):
        // Do something with the error
    }
}
